# 想定イシューリスト

このドキュメントは、MVP（Minimum Viable Product）開発に向けて想定されるタスクを管理します。

## #1 プロジェクト基盤の構築

- [ ] クリーンアーキテクチャに基づいたフォルダ構成の作成
- [ ] DIコンテナ（VContainerなど）の導入と設定
- [ ] Unity Input Systemのセットアップとアクションの定義

## #2 ゲームコアロジックの実装 (Domain層)

- [ ] タイマー機能（カウントダウン、時間の増減）の実装
- [ ] スコアとコンボの計算ロジックの実装
- [ ] バグ報告カードのデータ構造（Entity）の定義
- [ ] 「重要人物」の判定ロジックの実装
- [ ] ゲーム状態（プレイ中、結果表示など）を管理する仕組みの構築

## #3 ゲームプレイ画面の実装 (Presentation層)

- [ ] カード情報を表示するUIプレハブ（CardView）の作成
- [ ] カードを生成・管理するプーリングシステムの実装
- [ ] タイマー、スコア、コンボ数を表示するUIの実装
- [ ] プレイヤーの入力に応じてカードを左右に振り分ける機能の実装

## #4 ゲームフローの制御

- [ ] タイトル画面（ゲーム開始ボタン）の実装
- [ ] ゲームのメインループ（カードの逐次投入、タイムアップ判定）の実装
- [ ] リザルト画面（最終スコア、最大コンボ、ランク表示）の実装
- [ ] シーン遷移（Title → Game → Result）の管理

## #5 データ管理 (Infrastructure層)

- [ ] カードのテンプレートとなる`ScriptableObject`の作成
- [ ] ゲーム設定（制限時間、難易度など）を管理する`ScriptableObject`の作成
- [ ] ハイスコアをローカルに保存・読み込みする機能（PlayerPrefs）の実装

## #6 サウンドと演出

- [ ] 基本的な効果音（正解、不正解、ボタンクリック）の再生
- [ ] BGMの再生
